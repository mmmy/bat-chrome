BatChat WebSocket Monitor Settings

Information

This extension monitors WebSocket messages from wss://wsd.baaaat.com/ws on https://web.batchat.com/.

Messages are decoded from base64 and forwarded as JSON to your specified URL.

Message Format:

{
  "timestamp": "2025-01-23T14:32:50.123Z",
  "source": "bat-chat-websocket",
  "data": "decoded message content",
  "originalBase64": "base64 encoded message"
}
